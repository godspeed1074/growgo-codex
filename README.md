# GrowGo

GrowGo is a mobile-first web app built with plain HTML, CSS, JavaScript, and Leaflet.

## Project files

- `index.html` contains the app markup.
- `style.css` contains the responsive interface styling.
- `script.js` contains the map, stats, leaderboard, and app interaction logic.

## Running locally

Open `index.html` in a browser, or serve the folder with any static web server.
